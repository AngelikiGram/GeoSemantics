    GeoNarrator Spatial Engine — Guide
    ◎ Inspect (Place Lens)
    Real-time real-space character fingerprinting. Move the cursor over the map to view the 7 GNN semantic dimensions, scale attention (Micro/Meso/Macro), dominant archetype, and POIs. Click to lock/pin and optionally run live GNN inference.
    ↕ Compare (Semantic Diff)
    Multi-place comparative analysis. Click the map to set Location A and Location B. View their GNN embedding similarity, dimension-by-dimension deltas, and a natural-language comparison explanation.
    ~~ Morph (Semantic Morph)
    Embedding-space interpolation. Select Location A and Location B, and slide the tracker. Observe the GNN dimensions, archetypes, and UMAP coordinates continuously animate as the place identity merges.
    ◇ Search (Reverse Search)
    Fingerprint-based retrieval. Set custom slider targets for the 7 GNN dimensions to search and rank the closest matching precomputed locations in the database.
    ⊕ Sandbox (What-If Sandbox)
    Counterfactual planning. Lock a point and simulate placing synthetic POIs or deleting existing ones. Reruns GNN inference to predict exact character shifts.
    History (Temporal Time-Machine)
    Attic-year trajectory. Search any Austrian city/village (from all 7,627 places) or click the map in Live Fetch mode. Reconstructs past OSM records (since 2010) via attic queries to chart GNN character drift over time.
